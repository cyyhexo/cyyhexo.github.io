数据类型

int 整数

int => 4字节
unsigned int => 4字节
short int => 2字节
long int => 8字节

可以用sizeof(int)来获取字节数

定义的话

可以把int直接省略

输出
%hd => short int
%d => int
%ld => long int

关于 unsigned

数据是按照二进制的方式存储


如果是signed(默认)的话,第一位0表示的是+号,第一位1表示的是-号
因此 当溢出的时候符号会改变
如果是unsigned的话,只能表示从0开始的数!

float 浮点数

指数的表示 1.2E5 = 1.2e5 = 1.2×105

float => 4字节
double => 8字节

输出

  • %f 以十进制形式输出 float 类型
  • %lf 以十进制形式输出 double 类型
  • %e 以指数形式输出 float 类型，输出结果中的 e 小写
  • %E 以指数形式输出 float 类型，输出结果中的 E 大写
  • %le 以指数形式输出 double 类型，输出结果中的 e 小写
  • %lE 以指数形式输出 double 类型，输出结果中的 E 大写

注意
%f 和 %lf 默认保留6位小数 不足6位用0补齐,超过6位四舍五入
%n.mf 表示 域宽为 n,保留小数点后m位

  • %g 最多保留6位有效数字(不会补0)
  • %g 和 %lg 分别用来输出 float 类型和 double 类型，并且当以指数形式输出时，e小写。
  • %G 和 %lG 也分别用来输出 float 类型和 double 类型，只是当以指数形式输出时，E大写。

%n.mg 表示 域宽为 n,保留m位有效数字

int 与 float 转换

如果将int转换为 float 会自动在末尾补0
如果将float转换为int,会自动去掉小数点后面的数(不是四舍五入)

转义字符

\数字 表示 八进制形式的编码值
\x数字表示 十六进制形式的编码值

比如 a 的ASCII码 八进制为141 十六进制为61
那么可以表示为 \141 和 \x61

常见的表示

显示
转义字符 意义 ASCII码值（十进制）
\a 响铃(BEL) 007
\b 退格(BS) ，将当前位置移到前一列 008
\f 换页(FF)，将当前位置移到下页开头 012
\n 换行(LF) ，将当前位置移到下一行开头 010
\r 回车(CR) ，将当前位置移到本行开头 013
\t 水平制表(HT)  009
\v 垂直制表(VT) 011
\' 单引号 039
\" 双引号 034
\\ 反斜杠 092

数学运算

数学运算
  加法 减法 乘法 除法 求余数（取余）
数学 + - × ÷ 无
C语言 + - * / %

3/4 的结果为0 因为 0.75取整是0
如果 float a = 3/4 ,结果是 0.000000(先运算,再赋值)
3%-2=1 ; -3%-2=-1

a = a+1 => a += 1
其他同理

自加++ 和自减–

a = a+1 => a++
a = a-1 => a–

a++ 和 ++a的区别

比如 a = 10

  1. a1 = a++

    a1 = 10,a = 11

  2. a1 = ++a

    a1 = 11 ,a = 11

即 a++ 就是先将a的结果赋值给a1 然后再计算a+1
++a 就是先a+1,然后将结果赋值给a1
