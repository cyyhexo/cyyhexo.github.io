hexo --博客搭建

发表于2021-02-05|更新于2021-02-09|博客 hexo
|字数总计:553|阅读时长:2分钟|阅读量:

前言

搭建环境 win10 32位(虚拟机)

博客搭建 hexo + Github

如果下载慢,可在下载

CMD下Ctrl + C不是复制

方框是链接,可以点击

注册Github

  1. 注册,登录Github记住自己的 username
  2. 右上角+号 -> 点击 New repository
  3. Repository name 填写第一步的 username.github.io

    例如 我的username为123,就填写123.github.io

环境安装

  1. 安装 Node.js
  2. 安装 Git

    打开 Git Bash 输入

  3. 安装 Hexo

    打开 cmd 输入

    如果下载过慢可以先输入下面

建立Hexo

  1. 新建一个文件夹,用于存放博客
  2. 打开文件夹
    • 按住Shift 和鼠标右键
    • 点击在此处打开命令窗口
  3. 输入 等到加载成功,访问localhost:4000即可打开博客

配置公钥

获取公钥

需要将ssh-keygen设置为环境变量

  1. 打开cmd 输入

    一连输入三个 Enter
    得到 Your public key has been saved in /c/Users/Hexo/.ssh/id_rsa.pub.
    找到该文件 打开
    按 Ctrl + A 全选
    按 Ctrl + C 复制

填写公钥

  1. 打开 Github

    点击 New SSH Key
    填写 Title
    填写Key 粘贴

修改站点信息

在根目录下(默认为blog目录下),打开 _config.yml (可用记事本打开)
移到最下面,修改

点击

同步到Github

在根目录下(默认为blog目录下),运行cmd
输入

可能需要安装框架,可以不用安装最新的版本,看要求即可

然后会弹出Github登录 登录即可

成功部署

访问 https://username.github.io/ 即可!!!

可能需要等待几分钟同步...

域名设定

  1. 打开cmd 输入 获取ip,解析到域名
  2. 在根目录下的source 新建一个CNAME 写入域名

    即先创建一个.txt的文件,再改为CNAME

    一定要去掉后缀!!!!!!

  3. 同步到Github即可
